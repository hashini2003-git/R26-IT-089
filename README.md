# Oral-Health-Care-AI-Research

A research project aimed at early detection and prevention of oral cancer using data analysis and intelligent technologies to support better healthcare decisions.

## AI Voice Analysis Module

This module focuses on:
- Vocal clarity analysis
- Speech fluency detection
- Dysarthria detection
- Severity prediction
- Therapy progress monitoring

## Machine Learning Models Used
- Random Forest
- Gradient Boosting
- Voting Classifier
- SVM
- Logistic Regression
- XGBoost
